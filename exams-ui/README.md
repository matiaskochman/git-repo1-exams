exams-ui
==========

## Requirements

* [NodeJS](http://nodejs.org)

## Start the NodeJS server

___Context: a terminal in exams-ui folder___

__First time you launch the server:__
```
npm install
```

__Start the server:__
```
node server.js
```

## Display the app

___Context: a modern browser___

```
http://localhost:5000
```
